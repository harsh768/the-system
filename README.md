# The System — a life RPG

Turn your goals into **daily quests**. Clear them to earn XP, level up (rank E→S), build streaks, and spend gold on rewards you choose. A general-purpose, fully-customizable habit & life tracker — built as an offline PWA.

**Live:** https://harsh768.github.io/the-system/

## Features
- **Custom quests** — build your own or pick from a preset library (Fitness, Nutrition, Mind & Skill, Work & Focus, Self-care, Money). Set attribute, difficulty (XP/gold) and which days they repeat.
- **Keystones & streaks** — mark 2–3 quests that protect your streak, so missing the small stuff doesn't break the chain.
- **7 attributes + consistency web**, levels, ranks, and records.
- **Reward shop** — define real rewards, buy them with earned gold.
- **Money manager** (optional) — income, savings goal, budget categories, spend logging, donut breakdown, savings history. Any currency.
- **Calendar, activity & weight charts**, and a daily **learning log**.
- **Themes**, export/import backup, offline, installable, auto-updating.

## Install
Open the live URL → on iPhone tap **Share → Add to Home Screen**; on Android use **Install app**. Runs fullscreen and offline. All data stays on your device.

## Tech
Single static PWA (HTML/CSS/JS + service worker, no backend). Ready to wrap with Capacitor for the App Store / Google Play.
